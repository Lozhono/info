01

Модули

Всё, что нужно в одном месте.

[1]

Sender — рассылка в ЛС

Отправка сообщений пользователям.

  • ▸Получатели по @username или ID
  • ▸Telegram‑разметка текста: жирный, курсив, ссылки
  • ▸Прикрепление фото или файла
  • ▸Автостоп аккаунта при обнаружении спам‑блока
  • ▸Итоговая таблица: отправлено, ошибок, статус по каждому аккаунту
[2]

Group Sender — рассылка по группам

Массовая отправка сообщений в список групп.

  • ▸Список групп из текстового файла: @username или инвайт‑ссылки
  • ▸Если аккаунта нет в группе, он сам вступает перед отправкой
  • ▸Тот же движок текста: разметка, спинтакс, медиа
  • ▸Группы распределяются между выбранными аккаунтами
[3]

Parser — парсит участников

Собирает пользователей из групп как открытых так и закрытых.

  • ▸Открытые группы - прямой сбор полного списка участников
  • ▸Закрытые группы - парсинг по сообщениях в чате(количество выставляется)
  • ▸Автовступление в группу, если аккаунта там ещё нет
  • ▸Результат сохраняется в файл с названием группы (ID + username)
  • ▸Слишком маленькие группы пропускаются автоматически
[4]

Warmer — прогрев аккаунтов

Имитация живой активности перед началом работы с аккаунтом.

  • ▸Вступление в каналы и чаты из подготовленного списка
  • ▸Переписка между своими аккаунтами короткими фразами
  • ▸Вся активность укладывается в заданное время, до одного часа
  • ▸Итоговая статистика по каждому аккаунту
02

Менеджер аккаунтов

Добавление и вход

  • ▸Вход по API ID и API HASH:после нужно ввести номер и код подтверждения который прийдет на него, так же пароль двухэтапной авторизации, если он включён
  • ▸Импорт уже готовой сессии из папки tdata

Прокси на каждый аккаунт

  • ▸Индивидуальный прокси (SOCKS5 / HTTP) для каждого аккаунта

Статусы аккаунтов

  • Рабочий
  • Спам‑блок
  • Не работает
  • Не проверялся

Работа с несколькими аккаунтами

  • ▸Выбор диапазона аккаунтов (например «1,3‑5») или всех сразу
